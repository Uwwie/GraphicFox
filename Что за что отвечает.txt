Game:
	-db: папка с бибилиотекой h2. Без нее ничего не работает. Все хранится в бд h2.
	-helper: папка с библиотекой для работы с zip файлами (с паролем). Консоль без нее работает. Графика - нет, тк граф.режим берет из zip с паролем все ресурсы (картинки/музыка).
	-resources: папка с zip файлами в которых хранятся элементы граф. режима.
	-src:
		-repairkit: ЗАПУСТИТЬ 1 РАЗ ПЕРЕД ЗАПУСКОМ GAME. Он создает базу данных с таблицами и заполняет командами из .txt файлов из общей папки проекта.
			Почему нет защиты? repairkit не нужен пользователю для запуска game (пользователь получает уже готовую бд в папке). Это конструктор игры. Кт, поэтому repairkit остался для демонстрации.
		-классы:
			-ConsoleVariant: Консольная игра. Реагирует на ввод, берет данные из базы данных, обновляет ее после каждого выбора или команды save/quit.
			-Decryption: вынимает данные из архивов resources в автоматически создаваемую папку temp.
			-Game: мейн класс с которого начинается игра. Работает с методами Maintools.
			-GraphicVariant: Получает на вход данные игрока как и консольная версия. Запускает Decryption и SimpleGUI. Передает данные в SimpleGUI.
			-MainTools: отвечает за операции до выбора консольного/графического варианта игры. 
				Регистрирует/удаляет пользователя, загружает сейв/запускает НИ/запускает с кт, передает данные графическому или консольному варианту
			-SimpleGUI: swing gui. Отличается от консольного варианта кнопками вместо команд и работой с картинками/threads/музыкой/ссылками.
	-gametext.txt: команды для statement формата (int,string,string,int,int). Данные берутся для (номер нода, текст 1 варианта, текст 2 варианта, нод если в1 выбран, нод если в2 выбран)
	-quest.txt: текст квеста. Номер линии = номеру нода из gametext.txt. Он не загружается в бд, его нельзя удалять (нужен даже после repairkit запуска). Остальные txt все загружаются.
	-references.txt: команды для statement (int, string, float). Данные берутся для (номер нода, название песни, громкость). Загружается в бд.
	-secretcode.txt: команды для statement (int,int,string). Данные берутся для (номер нода, есть ли секрет(1/0), ссылка). В swing нельзя запускать mp4, поэтому так (или fx включать а тут уже 2 бибилиотеки в кт).
